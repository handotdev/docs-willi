The appearance settings in your knowledge base let you shape every visual detail of the experience — from color scheme and contrast to the fonts used in both the interface and the editor. You can reach all of these options under **Settings → Themes**, and most changes render instantly so you can preview them in real time.

## Light and Dark Mode

Toggle between light and dark mode using the **sun/moon icon** in the top-right toolbar, or go to **Settings → Themes → Color Mode** for the full set of options:

- **Light** — A bright, high-contrast surface suited to well-lit environments.
- **Dark** — A dark background that reduces eye strain in low-light conditions.
- **Auto** — Follows your operating system's appearance setting and switches automatically at sunrise/sunset.

## Color Themes

Color themes change the accent colors, backgrounds, and UI chrome across the entire app. Choose a theme under **Settings → Themes → Color Theme**.

## Font Options

### Interface Font

Controls the typeface used in the sidebar, menus, and all UI elements outside the note editor. Choose from:

- **System Default** — Uses the default font of your operating system (San Francisco on macOS, Segoe UI on Windows).
- **Inter** — A versatile, screen-optimised sans-serif that looks sharp at any size.
- **Serif** — A classic serif option for a more editorial, print-like feel.

### Editor Font

Controls the typeface used inside the note editor itself. Choose from:

- **Monospace** — Recommended when writing lots of code blocks or structured Markdown tables; every character is the same width.
- **Proportional** — A comfortable reading font for prose-heavy notes.

### Font Size

Use the **Font Size** slider under **Settings → Themes → Typography** to set your preferred base size between **12 px** and **20 px**. The slider adjusts both the interface and editor fonts together.

## Focus / Zen Mode

Pressing <kbd>Cmd/Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>F</kbd> activates Focus mode, which hides the sidebar, toolbar, and all other chrome — leaving only your note on screen. Press the same shortcut again, or press <kbd>Esc</kbd>, to return to the normal view.

<Tip>
  Use a monospace editor font when writing lots of code or structured Markdown.
</Tip>
