Keyboard shortcuts let you navigate, write, and switch views without ever reaching for your mouse, keeping you in a focused flow state. You can see every available shortcut at a glance by pressing <kbd>?</kbd> anywhere in the app, or by going to **Settings → Shortcuts** for the full reference and customization panel.

## Navigation Shortcuts

Use these shortcuts to move around the app and create notes quickly.

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl` + `K` | Open search |
| `Cmd/Ctrl` + `N` | Create a new note |
| `Cmd/Ctrl` + `Shift` + `N` | Open Quick Capture (lightweight inline note) |
| `Cmd/Ctrl` + `,` | Open Settings |

## Editor Shortcuts

Use these shortcuts while writing or editing a note.

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl` + `B` | Toggle **bold** formatting |
| `Cmd/Ctrl` + `I` | Toggle *italic* formatting |
| `Cmd/Ctrl` + `[` or `]` | Insert or toggle a checklist item |
| `Cmd/Ctrl` + `` ` `` | Wrap selection in inline `code` |
| `Cmd/Ctrl` + `S` | Save the current note |

## View Shortcuts

Use these shortcuts to change how the app is displayed.

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl` + `\` | Toggle the sidebar open or closed |
| `Cmd/Ctrl` + `Shift` + `F` | Enter or exit Focus mode |
| `Cmd/Ctrl` + `G` | Open Graph view |

## Customizing Shortcuts

You can remap any shortcut to a key combination you prefer.

<Steps>
  <Step title="Open the Shortcuts panel">
    Go to **Settings → Shortcuts** to see the full list of actions and their current bindings.
  </Step>
  <Step title="Select a shortcut to change">
    Click on the shortcut row for the action you want to remap. The row will highlight and show a **Press new shortcut** prompt.
  </Step>
  <Step title="Press your new key combination">
    Hold the keys you want to use (e.g., <kbd>Cmd/Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd>) and release. The new binding saves automatically.
  </Step>
  <Step title="Resolve conflicts (if any)">
    If the combination is already in use, the app will warn you and ask whether to reassign it or keep the existing binding.
  </Step>
</Steps>

<Note>
  On Windows and Linux, replace Cmd with Ctrl for all shortcuts.
</Note>

<Tip>
  Press `?` anywhere in the app to open the shortcut cheat sheet overlay.
</Tip>
